# CGPA Calculator (C++)

## Description
This project is a CGPA Calculator developed using C++ as part of the
CodSoft Internship program. The program calculates the CGPA based on
the grades or marks entered by the user.

## Technology Used
- C++

## Features
- Takes user input for subjects
- Calculates total CGPA
- Displays final CGPA clearly

## How to Run
1. Open the project in any C++ compiler (Dev C++, CodeBlocks, or VS Code).
2. Compile the source file.
3. Run the program.
4. Enter the required marks/grades to calculate CGPA.
